Welcome to ng-ui2qp!

This guide introduces you to the essentials of ng-ui2qp by walking you through the installation, configuration and, a simple use case.

Pre-requisites

Before using the library be sure the next requisites are filled out:

  • Angular Router Service must be available. This is only required if you use the Ui2QpRouter(Default router adapter provided by us).
  • ReactiveFormsModule must be imported and accessible.

Create a Ui2QpGroup and its controls, like it's done when using ReactiveForms

Instantiate a Ui2QpRoot object and link it to the created model

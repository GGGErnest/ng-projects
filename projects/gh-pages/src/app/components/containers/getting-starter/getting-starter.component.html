Getting Started

Pre-requisites

Before using the library are some requisites that should be fill here they are:

  • Angular Router Service must be available. This is only required if you use the Ui2QpRouter.
  • ReactiveFormsModule must be imported and accessible.

How to use.

Inject the Ui2QpBuilder service

Create a Ui2QpGroup and its controls, like you'll do when creating a ReactiveForm

Instantiate a Ui2QpRoot object and link the created model with it

It's also possible to do the previous step and this one at once, here is how:

WARNING: The Ui2QpRoot will holds a subscription to the values changes observable of the model if the "autoUpdating" settings was enabled, and this means that you most execute the "destroy" method in the Ui2QpRoot within the "onDestroy" of the component where was created, this is needed in order to avoid memory leaks.

Bind the model with the template, like it's done with ReactiveForms

v2.0.0

Changes from 1.0.3:

  • The lib was updated to Angular 10
  • Updated dependencies to their last versions
  • Added support for Angular Schematics
  • Added Logging to all the library features. This can be enabled in the configurations.
  • Added support to Qps name definition, which means the name of Qps can be defined, in previous version this wasn't possible, the names were defined depending on the model.
  • Added crypto serializer and de-serializer. This allows ciphering Qps values so they aren't readable from the URL.
  • Some classes and exported types were renamed. This cause some incompatibilities with previous versions.
  • Some known bugs were fixed.
  • Added Template-Driven approach.
  • Enhanced Ui2QpRoot so the model doesn't necessary needs to be passed in on the constructor, can eb linked to the Ui2QpRoot instance later.
